# ItEra-v1
This is the first version of ItEra Studio that I am developing as a mini project (4 weeks) for my Junior Seminar. 

ItEra is a structured AI-assisted critique system for visual art.
This project explores how multimodal LLMs can support artistic growth
through intention-aware feedback.
